# Celebrate Hub

A web application that connects customers with service providers for various events and celebrations.

## Features

*   User authentication (Login, Register)
*   Separate dashboards for Customers, Service Providers, and Admins.
*   Service listings and management.
*   Booking and appointment management.
*   Customer reviews and ratings.
*   Admin panel for user and service management.

## Technologies Used

### Frontend

*   React
*   React Router
*   Axios

### Backend

*   Node.js
*   Express.js
*   MongoDB
*   Mongoose

## Installation

1.  Clone the repository:
    ```bash
    git clone https://github.com/affan-altaie/celebrate.git
    ```
2.  Install server dependencies:
    ```bash
    cd Celebratehub-Main/server
    npm install
    ```
3.  Install client dependencies:
    ```bash
    cd ../client
    npm install
    ```
4.  Create a `.env` file in the `server` directory with your MongoDB connection string:
    ```
    MONGODB_URI=your_mongodb_connection_string
    ```
5.  Start the server:
    ```bash
    npm start
    ```
6.  Start the client:
    ```bash
    npm start
    ```

## Usage

*   Access the application at `http://localhost:3000`.
*   Register as a customer or service provider.
*   Browse and book services.
*   Manage your profile and bookings from your dashboard.
